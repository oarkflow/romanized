## 📚 Complete Vowel Guide (स्वर बर्ण)

### Basic Vowels

Type the complete vowel alphabet:
```
Type: a aa i ee u oo e ai o au ri
Get:  अ आ इ ई उ ऊ ए ऐ ओ औ ऋ
```

### Detailed Vowel Mappings

| Roman | Devanagari | Matra | Example |
|-------|------------|-------|---------|
| a | अ | (inherent) | **ka** → क |
| aa, a: | आ | ा | **kaa** → का |
| i | इ | ि | **ki** → कि |
| ee, ii | ई | ी | **kee** → की |
| u | उ | ु | **ku** → कु |
| oo, uu | ऊ | ू | **koo** → कू |
| e, ei | ए | े | **ke** → के |
| ai | ऐ | ै | **kai** → कै |
| o, oi | ओ | ो | **ko** → को |
| au | औ | ौ | **kau** → कौ |
| ri | ऋ | ृ | **kriti** → कृति |

### Inherent 'a' vs Long 'ā'

In Devanagari, consonants have an inherent 'a' sound:
- `sa` = स (inherent a)
- `saa` = सा (long ā with matra)
- `paisa` = पैस (pais + inherent a)
- `paisaa` = पैसा (pais + long ā)

## 📝 Diacritics (बिन्दु)

### Anusvara (अनुस्वार)

**Anusvara (ं)** - Use `~m` or `m~` after a vowel:
- `jhai~m` = झैं
- `bhanejhai~m` = भनेझैं
- `kam~` → कं

### Candrabindu (चन्द्रबिन्दु)

**Candrabindu (ँ)** - Use `~n` or `n~` after a vowel:
- `jhai~n` = झैँ
- `bhanejhai~n` = भनेझैँ
- `hun~` → हुँ

### Other Diacritics

| Roman | Devanagari | Name | Example |
|-------|------------|------|---------|
| h~, ~h | ः | Visarga | **namah~** → नमः |
| .a, ' | ऽ | Avagraha | **de.a** → देऽ |

## 🎯 Special Rules

### 1. Auto-halant for consonant clusters
No need to type halant manually - it's inserted automatically:
- **garchu** → ग + र + ् + च + ु = गर्चु
- **ramro** → रम्रो

### 2. Manual halant with ^
Use `^` to force a halant:
- **k^sha** → क्ष
- **r^ti** → र्ति

### 3. Context-aware dh
The system intelligently handles 'dh':
- **padhchhu** → पढ्छु (dh before ch/chh → ढ)
- **dhan** → धन (dh otherwise → ध)

### 4. Om symbol
Type **om** in isolation (surrounded by spaces or punctuation) → ॐ

### 5. Proper noun lexicon
1,225+ cities and districts are recognized automatically:
- **Kathmandu** → काठमन्डु
- **Pokhara** → पोखरा

## 💡 Tips for Best Results

1. **Use double vowels for long sounds**: `aa`, `ee`, `oo` instead of `a`, `i`, `u`
2. **Capitalize for retroflex consonants**: `T`, `Th`, `D`, `Dh`, `N`, `Sh`
3. **Use ~ for diacritics**: `~m` for ं, `~n` for ँ, `~h` for ः
4. **Let auto-halant work**: Don't manually add halants for clusters
5. **Check proper nouns**: Many common place names are auto-recognized

## ⚠️ Known Limitations

1. **Romanization ambiguity**: Multiple romanizations possible for the same Nepali text
2. **Inherent vowel detection**: Context-dependent in reverse mode
3. **Homonyms**: `ee`/`ii` both map to ई in forward direction
4. **Rare clusters**: Some uncommon Sanskrit clusters may need manual addition
