💡 How to Type Special Characters:

  • • Basic consonants: kh → ख, gh → घ, D → ड
  • • With matras: kha → खा, Da → डा (no conflict!)
  • • Nukta variants: kh. → ख़, gh. → ग़ (use dot!)
  • • Dravidian: .r → ऱ, .l → ळ (dot first!)
  • • Vedic accents: '1 → ॑, '2 → ॒

Priority 2 Vedic Accent Marks

Note: Vedic accents are combining marks that appear above or below characters. They are essential for proper Vedic Sanskrit pronunciation.
Input Mark Name Unicode
'1 or ^1 ॑ Udatta (high pitch) U+0951
'2 or _1 ॒ Anudatta (low pitch) U+0952
'3 or `1 ॓ Grave accent U+0953
'4 or '1 ॔ Acute accent U+0954
'5 ᳐ Svarita U+1CD0

+ 12 more Vedic accent marks (U+1CD1-U+1CDA) for complete Sanskrit support
